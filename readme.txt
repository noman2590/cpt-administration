Simple and plugin to manage custom post types. Simple but flexible.

== Description ==

This is a simple plugin with that you can easily manage custom post types in WordPress. It has the following features:

1. Register custom post types with a simple form.
2. Edit custom post types.
3. Delete registered custom post types.
4. Enable/disable categories and tags support.
5. Enable/disable support for Title, Editor, Featured Image, Excerpt, Trackbacks, Custom Fields, Comments, Revisions, Author, Page Attributes, Post Format.
5. Unlimited custom post types.


= Privacy notices =

With the default configuration, this plugin, in itself, does not:

* write any user personal data to the database;
* send any data to external servers;
* use cookies.

== Installation ==

1. Upload the entire `my-custom-post-types` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
3. You will find **CPT List** menu in your WordPress admin screen where you can manage custom post types.
